# Portal de Inscrição ENEM

Este projeto simula um portal de inscrição do ENEM, desenvolvido como atividade prática de Front-End. O sistema contém telas de login e inscrição, com validação de dados utilizando HTML, CSS e JavaScript.

## 📌 Objetivo
Desenvolver interfaces funcionais com validação de formulários, manipulação do DOM e navegação entre páginas.

## 💻 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 🧾 Funcionalidades
- Tela de login com validação de e-mail e senha
- Tela de inscrição com formulário completo do candidato
- Validação de campos obrigatórios
- Verificação de e-mail (com @)
- Verificação de senha (mínimo de 8 caracteres)
- Confirmação de senha
- Exibição de mensagens de erro na tela
- Redirecionamento entre páginas

## 📄 Estrutura do projeto
- login.html → Tela inicial de login
- cadastro.html / inscricao.html → Formulário de inscrição
- style.css → Estilização da tela de login
- inscricao.css → Estilização da tela de inscrição
- script.js → Validação do login
- inscricao.js → Validação da inscrição

## 🚀 Como executar
1. Baixe ou clone o repositório
2. Abra o arquivo `login.html` no navegador
3. Navegue entre as páginas usando os botões do sistema
